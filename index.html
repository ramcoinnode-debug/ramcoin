🚀 О проекте

🖥️ CPU-only майнинг

Майнит любой домашний ПК с 4 ГБ ОЗУ. Никаких видеокарт, никаких ASIC. Только процессор и оперативная память.

🛡️ Защита от серверов

Встроенный детектор серверных CPU. Штраф 75% для Xeon, EPYC, Threadripper. Всё по-честному — домашние ПК в приоритете.

⚡ Соло-майнинг

Никаких пулов. Нашёл блок — забрал всю награду. Ни с кем не делишься. Как Bitcoin в 2009 году.

💰 Честный старт

0 премайна. 0 ICO. 0 инвесторов. Разработчик майнит наравне со всеми с первого блока. Код открыт — проверь сам.

🔍 Полная прозрачность

Вся история блокчейна хранится в открытой базе данных SQLite. Любой может проверить все блоки, все транзакции, все балансы. Никаких скрытых начислений. Всё как на ладони.

📂 База данных: blockchain_v7.db (SQLite3, можно открыть любым просмотрщиком)

⛏️ Как начать майнить

📥 Быстрый старт

  1. Установи Python 3.10+
  2. pip install -r requirements.txt
  3. Скачай файлы с GitHub
  4. Запусти: python miner.py
  5. Кошелёк создастся автоматически

⚡ Производительность

Хешрейт зависит от L3-кэша процессора:

  • 🖥️ Ryzen 5 5600X (32MB L3): ~30-50 Ram/s
  • 💻 Ryzen 5 5500U (8MB L3): ~8-15 Ram/s
  • 🖥️ i5-12400 (18MB L3): ~20-35 Ram/s
  • 🖥️ i3-12100 (12MB L3): ~10-20 Ram/s

Майнер сам определит ваш процессор и подберёт оптимальные настройки.
