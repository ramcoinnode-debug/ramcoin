# RAMCOIN v11 — Parasite Protocol

Децентрализованная криптовалюта без серверов, без белых IP, без доменов.

## Особенности
- Блоки каждые 20 секунд
- Халвинг каждые 876 000 блоков
- Награда: 10 RAM → 5 → 2.5 → ...
- Эмиссия: ~17.5 млн RAM
- Сжигание: 1% с каждого блока
- Транспорт: GitHub Gist (устойчив к блокировкам)
- Нет майнинга, нет затрат электричества

## Быстрый старт

pip install aiohttp cryptography lz4
python ramcoin_v11_node.py

## Кошелёк

pip install pycryptodome
python ramcoin_v11_wallet.py
